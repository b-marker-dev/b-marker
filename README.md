## English Version

Project Overview

B-Marker is a profit-sharing management framework designed specifically for time/sound tool developers. It adopts a layered model combining open-source core functionality and commercial value-added services, aiming to balance the openness of open-source collaboration and the sustainability of commercial operations. The framework provides clear profit-sharing rules, standardized contribution assessment mechanisms, and flexible authorization models, enabling developers to jointly build high-quality music tools while fairly sharing the commercial benefits brought by the project.

Module Division

1. bmarker-core (Open Source, MIT License): The foundational core module of the framework, providing core algorithms and basic functional components related to music tool development (e.g., beat detection, audio data processing, profit-sharing data modeling). It is completely open-source and free for all developers to use, modify, and integrate, laying a solid foundation for collaborative construction.

2. bmarker-tools (Commercial Authorization): A toolset built on the core module, focusing on practical integration solutions for music tools (e.g., third-party platform docking, enterprise-level customization tools, batch processing plugins). It is free for individual developers and requires paid authorization for commercial use by enterprises.

3. bmarker-pro (Subscription License): The premium version of the framework, offering advanced value-added functions such as cloud synchronization, professional audio analysis, exclusive sound effect libraries, and customized profit-sharing data dashboards. It adopts a tiered subscription model (individual/team/enterprise plans) to meet the diverse needs of professional developers and commercial teams.

Core Advantages

- Open & Transparent Profit-Sharing: All revenue sources, sharing ratios, and settlement cycles are fully disclosed. Profit-sharing is strongly tied to contribution levels (code development, testing, documentation, promotion, etc.), ensuring fair distribution of benefits.

- Flexible Layered Authorization: The open-source core ensures the openness and scalability of the project, while commercial modules guarantee the sustainability of the project's operations and the income of contributors.

- Low Contribution Threshold: Supports both code and non-code contributions (e.g., documentation writing, community support, promotion). New contributors can participate in profit-sharing after completing one settlement cycle without a lock-up period.

- Standardized Collaboration Process: Equipped with complete contributor agreements, PR templates, and issue handling mechanisms, reducing collaboration costs and improving development efficiency.

Target Users

Music tool developers, audio technology enthusiasts, independent developers focusing on music software, and enterprises needing customized music tool solutions.

License & Collaboration

- bmarker-core: MIT License (open-source, free for all use)

- bmarker-tools: Commercial License (free for individuals, paid for enterprises)

- bmarker-pro: Subscription License (tiered pricing for individuals/teams/enterprises)

We welcome developers worldwide to participate in the construction of the project. For contribution guidelines and profit-sharing details, please refer to the CONTRIBUTING.md and profit-sharing/framework.md documents in the repository.
